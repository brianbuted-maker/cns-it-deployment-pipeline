Student Worker Documentation Tool

How to Use This Tool

1. Select your task type (Imaging, Pickup, or New Deploy)
2. Fill in the device / request details from the INC or TASK
3. Review the auto-generated outputs on the final screen
4. Click Copy next to each output and paste into Planner / ServiceNow
5. Check off the post-copy checklist, then hit Start New Task
Teams Access: This app is pinned as a tab in your Deployments - Students channel in CNS IT Teams. Open Teams → channel → click the CNS Deployment Docs tab.
